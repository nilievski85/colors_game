# colors_game
An addictive game, build with HTML, CSS and JavaScrip using jQuery framework

 Usage: 
 CG = new ColorsGame(initial_options);
 CG.build(options);
 
 default options: {
   width:  5,
   height: 5,
   colors_count: 5,
   cont:   $('#game_cont'),
   tail_tmp: '<div class="cc c_$color" id="$id"></div>',
   autobuild: 0
 }

Start index.html to view the demo.
